# Node.js Examples

Native `fetch` (Node 18+). No dependencies for the basic scripts.

```bash
export DIAMOND_KEY="your_key_from_signup.php"
node 01-predictions-today.js
```

| Script | What it does |
|--------|--------------|
| `01-predictions-today.js` | Print today's predictions sorted by confidence |
| `02-value-bets-alert.js` | Find 5%+ edges vs Vegas; post to Slack/Discord webhook |
| `03-team-matchup.js` | Single-matchup prediction with feature breakdown |
